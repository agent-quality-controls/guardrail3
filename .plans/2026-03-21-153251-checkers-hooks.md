## Rule count: 19 rules across 3 sub-checkers

| Checker | Rules | Input |
|---------|-------|-------|
| hooks/shared | 8 | pre-commit script content |
| hooks/rs | 6 | pre-commit script content (Rust steps) |
| hooks/ts | 5 | pre-commit script content (TS steps) |

All three sub-checkers read the same file (pre-commit hook script) but check for different patterns. The shared checker validates structure, rs/ts checkers validate language-specific tool steps.
