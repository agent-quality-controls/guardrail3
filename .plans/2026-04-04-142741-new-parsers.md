# Two new parser crates: cargo-config-toml and guardrail3-toml

## cargo-config-toml

Parser for `.cargo/config.toml` files.

### Schema (from Cargo reference)

```toml
[build]
target = "x86_64-unknown-linux-gnu"
jobs = 4
rustflags = ["-C", "target-cpu=native"]

[env]
MY_VAR = "value"
MY_VAR = { value = "value", force = true, relative = false }

[target.x86_64-unknown-linux-gnu]
linker = "clang"
rustflags = ["-C", "link-arg=-fuse-ld=lld"]

[registries.my-registry]
index = "https://my-registry.example.com"

[registry]
default = "crates-io"

[net]
retry = 2
git-fetch-with-cli = true

[http]
proxy = "..."
timeout = 30

[alias]
b = "build"
```

### Used by guardrail3

Currently only the clippy family reads it, checking:
- `[env].CLIPPY_CONF_DIR` — forbidden override that bypasses clippy.toml routing

### Types

```rust
pub struct CargoConfig {
    pub build: Option<BuildConfig>,
    pub env: BTreeMap<String, EnvValue>,
    pub target: BTreeMap<String, TargetConfig>,
    pub registries: BTreeMap<String, RegistryConfig>,
    pub registry: Option<RegistryDefaultConfig>,
    pub net: Option<NetConfig>,
    pub http: Option<HttpConfig>,
    pub alias: BTreeMap<String, Value>,
    #[serde(flatten)]
    pub extra: BTreeMap<String, Value>,
}

#[serde(untagged)]
pub enum EnvValue {
    Simple(String),
    Detailed(EnvValueDetail),
}

pub struct EnvValueDetail {
    value: String,
    force: Option<bool>,
    relative: Option<bool>,
    #[serde(flatten)]
    extra: BTreeMap<String, Value>,
}
```

## guardrail3-toml

Parser for `guardrail3.toml` — our own config format.

### Schema (from current usage)

```toml
version = "0.1"

[profile]
name = "service"  # or "library"

[rust]
workspace_root = "apps/guardrail3"
excluded_paths = ["tests/fixtures", "tests/snapshots", "legacy"]

[rust.checks]
fmt = true
toolchain = true
clippy = true
deny = true
cargo = true
code = true
hexarch = true
deps = true
garde = true
test = true
release = true
hooks_shared = true
hooks_rs = true

[rust.apps.<name>]
type = "service"
profile = "service"

[rust.apps.<name>.checks]
hexarch = false

[rust.packages]
type = "library"

[rust.packages.checks]
hexarch = false
garde = false

[rust.packages.deps]
allowed_deps = ["serde", "toml"]

[[escape_hatches]]
family = "cargo"
file = "apps/guardrail3/Cargo.toml"
kind = "lint_allow"
selector = "clippy:redundant_pub_crate"
reason = "..."
```

### Types

```rust
pub struct Guardrail3Config {
    pub version: Option<String>,
    pub profile: Option<ProfileConfig>,
    pub rust: Option<RustConfig>,
    #[serde(default)]
    pub escape_hatches: Vec<EscapeHatchConfig>,
    #[serde(flatten)]
    pub extra: BTreeMap<String, Value>,
}

pub struct ProfileConfig {
    pub name: Option<String>,
    #[serde(flatten)]
    pub extra: BTreeMap<String, Value>,
}

pub struct RustConfig {
    pub workspace_root: Option<String>,
    #[serde(default)]
    pub excluded_paths: Vec<String>,
    pub checks: Option<RustChecksConfig>,
    #[serde(default)]
    pub apps: BTreeMap<String, AppConfig>,
    pub packages: Option<PackagesConfig>,
    #[serde(flatten)]
    pub extra: BTreeMap<String, Value>,
}

pub struct RustChecksConfig {
    pub fmt: Option<bool>,
    pub toolchain: Option<bool>,
    pub clippy: Option<bool>,
    pub deny: Option<bool>,
    pub cargo: Option<bool>,
    pub code: Option<bool>,
    pub hexarch: Option<bool>,
    pub deps: Option<bool>,
    pub garde: Option<bool>,
    pub test: Option<bool>,
    pub release: Option<bool>,
    pub hooks_shared: Option<bool>,
    pub hooks_rs: Option<bool>,
    #[serde(flatten)]
    pub extra: BTreeMap<String, Value>,
}

pub struct AppConfig {
    #[serde(rename = "type")]
    pub app_type: Option<String>,
    pub profile: Option<String>,
    pub checks: Option<RustChecksConfig>,
    #[serde(flatten)]
    pub extra: BTreeMap<String, Value>,
}

pub struct PackagesConfig {
    #[serde(rename = "type")]
    pub package_type: Option<String>,
    pub checks: Option<RustChecksConfig>,
    pub deps: Option<PackageDepsConfig>,
    #[serde(flatten)]
    pub extra: BTreeMap<String, Value>,
}

pub struct PackageDepsConfig {
    #[serde(default)]
    pub allowed_deps: Vec<String>,
    #[serde(flatten)]
    pub extra: BTreeMap<String, Value>,
}

pub struct EscapeHatchConfig {
    pub family: Option<String>,
    pub file: Option<String>,
    pub kind: Option<String>,
    pub selector: Option<String>,
    pub reason: Option<String>,
    #[serde(flatten)]
    pub extra: BTreeMap<String, Value>,
}
```
