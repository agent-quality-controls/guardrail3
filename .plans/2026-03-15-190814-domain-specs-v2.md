## Conventions

CheckResult struct:
```rust
CheckResult { id: String, severity: Severity, title: String, message: String, file: Option<String>, line: Option<usize> }
```

Check functions: `fn check_xxx(workspace_root: &Path, ...) -> Vec<CheckResult>`
Or push-style: `fn check_xxx(..., results: &mut Vec<CheckResult>)`

Tests: `#[cfg(test)] mod tests` in same file. Per check: one negative test (violation → finding) + one positive test (no violation → no finding). Inventory/Info-only checks get at least one test proving they emit.
