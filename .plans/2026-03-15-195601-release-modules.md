# Add release-plz and cliff.toml module support

**Date:** 2026-03-15 19:56
**Task:** Create release.rs module, register it, wire into generate and init commands

## Goal
guardrail3 can generate and scaffold release-plz.toml and cliff.toml from embedded modules.

## Approach

1. Create `src/modules/release.rs` with RELEASE_PLZ_TOML and CLIFF_TOML constants
2. Register in `src/modules/mod.rs` (add `pub mod release` + push to all_modules)
3. Update `src/commands/generate.rs` — generate these files for service/monorepo profiles
4. Update `src/commands/init.rs` — scaffold these files during init

## Files to Modify
- `src/modules/release.rs` — new file with module constants
- `src/modules/mod.rs` — register release modules
- `src/commands/generate.rs` — generate release configs
- `src/commands/init.rs` — scaffold release configs
