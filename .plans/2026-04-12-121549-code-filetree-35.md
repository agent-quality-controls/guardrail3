- Migrate the only remaining live package target in the code family, old app RS-CODE-35, into a package file-tree lane as RS-CODE-FILETREE-35.

- Add failing rule-local and ingestion pipeline tests for the structural-cap rule.

- Treat RS-CODE-35 as a file-tree rule because it checks root structural counts, not config contents or source contents.
