# Fix 5 parser bugs found by adversarial tests

**Date:** 2026-03-17 23:44

## Bugs

1. **Multiline entries** — diff.rs collect_toml_entries is line-based. Fix: join continuation lines before matching.
2. **No-space syntax** — checks `{ path =` but not `{path=`. Fix: normalize whitespace before matching.
3. **Cross-section dedup** — flat BTreeSet. Fix: prefix entries with section name (methods: vs types:).
4. **Override duplicates base** — generate.rs injects overrides verbatim. Fix: dedup entries after merge.
5. **Malformed override** — no validation. Fix: validate TOML syntax of override content before injecting.

## Files
- commands/diff.rs — bugs 1, 2, 3 (custom entry detection)
- commands/generate.rs — bugs 4, 5 (override injection)
