# Summary

Implemented the standalone TypeScript Astro integration package `g3ts-astro-llms`. The package validates explicit config with Zod and writes deterministic `llms.txt` during Astro's `astro:build:done` hook.

# Decisions Made

- Kept the package standalone with package-local `package.json`, lockfile, `tsconfig`, tests, and built `dist` output because existing TypeScript packages in this repo use package-local npm structure.
- Used a single `astro:build:done` hook and writes only `llms.txt`.
- Accepted optional `contentAdapters` as explicit config, but did not infer or crawl content folders.
- Sorted sections and links deterministically and normalized generated URLs to absolute site URLs.
- Preserved meaningful link query strings while dropping hash fragments.
