# Add guardrail3-check-types shared types package

**Date:** 2026-04-04 14:03

## Summary
Shared types for guardrail3 check packages: GrdzCheckResult, GrdzSeverity,
GrdzProfile (Application/Library). Zero dependencies, zero clippy warnings,
guardrail-compliant. This is the foundation for extracting family content
checkers into independent packages.
