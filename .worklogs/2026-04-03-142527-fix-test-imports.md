# Fix test imports after facade cleanup

**Date:** 2026-04-03 14:25

## Summary
Fixed test compilation across all non-hexarch families. The facade cleanup
left broken imports in tests/mod.rs and test files. Three categories of fix:
1. tests/mod.rs `pub(crate) use super::X` → `pub(crate) use helpers::X`
2. tests/mod.rs `use super::helpers::X` → `use helpers::X` (self-reference)
3. test files `super::helpers::check(` → `super::super::check(` (check is in rule, not helpers)
4. garde: `pub(crate) mod helpers;` + cross-module re-export for canonical_clippy_toml
5. hexarch: `#![allow(...)]` → `#[allow(...)]` (inner → outer attribute)
