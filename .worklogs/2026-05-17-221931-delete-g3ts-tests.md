# Delete G3TS Tests

## Summary

Removed the remaining G3TS test-only directories, Rust assertion crates, Rust test-module wiring, and npm package test scripts now that G3TS rule behavior is covered by fixture replay.

## Decisions Made

- Deleted only G3TS-scoped test surfaces under `apps/guardrail3-ts` and `packages/ts`.
- Removed G3TS `crates/assertions` workspace members and deleted assertion crate dev-dependencies.
- Removed `#[cfg(test)]` module wiring from G3TS Rust source files.
- Removed npm `test` and `pretest` scripts that referenced deleted `tests/*.test.ts` files.
- Changed npm `prepublishOnly` scripts from `npm test` to `npm run build`.
- Removed deleted `tests/**/*.ts` paths from G3TS package `tsconfig*.json` files.
