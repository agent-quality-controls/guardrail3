# Fix mangled mod.rs files from cleanup script

**Date:** 2026-04-03 13:28

## Summary
The cleanup script partially removed #[cfg(test)] functions from unsplit
rule mod.rs files — stripped the attribute and signature but left the body,
causing syntax errors. Reverted 39 mangled files to their pre-cleanup state.
Fixed 2 hexarch mod.rs files (22, 23) that were committed broken.

## Next steps
The unsplit rules need a different approach — full split (production→rule.rs)
not just #[cfg(test)] removal. The cleanup script should only run on rules
that already have rule.rs.
