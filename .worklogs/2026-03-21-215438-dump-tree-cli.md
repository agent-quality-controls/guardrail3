# Add dump-tree CLI command

**Date:** 2026-03-21

## What

`guardrail3 dump-tree [path]` — outputs the full ProjectTree as pretty-printed JSON to stdout. Shows structure (every dir + children) and content (cached config files).

## Files changed

- `crates/adapters/inbound/cli/cli.rs` — added `DumpTree` variant to `Commands` enum
- `crates/main.rs` — added `handle_dump_tree` handler
