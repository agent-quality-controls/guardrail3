# Ban grep-* Rust crates

**Date:** 2026-03-19 19:12
**Scope:** deny.toml

## Summary
Added grep-cli, grep-regex, grep-matcher to deny.toml banned crates. These are the ripgrep component crates that would indicate regex-based parsing.

## Note on TypeScript RegExp
Built-in RegExp is a language primitive — banning it universally would be overreach. It's legitimate for input validation, string parsing, test assertions. The ban targets exotic regex libraries (xregexp, regexp-tree) that indicate heavy text processing where structured parsers should be used instead.
