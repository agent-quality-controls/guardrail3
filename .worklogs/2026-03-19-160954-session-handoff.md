ALL config file fields are `Vec<PathBuf>` (not Option) — no instances silently dropped in monorepos.

### 2. Coverage Map Engine (`src/commands/coverage/engine.rs`)
Generic engine with `CoverageTool` trait. Each tool implements: name, resolution_description, config_files, source_dirs, parse_details, walks_up.

Engine does:
- Directory-based coverage resolution (not crate-based, not file-based — directory is the coverage unit)
- Walk-up simulation per source directory for walk-up tools
- Collapse of uncovered directories to top-level ancestors (stops at one level below project root)
- Shadow detection: configs nested inside another config's coverage area get `is_shadow: true` + `shadows: "<parent>"`, parent gets `shadowed_by: [{ path, steals }]`
- JSON output only (text tree format removed)

## How to Run Coverage Maps

```bash
# All tools have --<tool> flag
cargo run -- map --clippy /path/to/project
cargo run -- map --deny /path/to/project
cargo run -- map --rustfmt /path/to/project
cargo run -- map --rust-toolchain /path/to/project
cargo run -- map --eslint /path/to/project
cargo run -- map --tsconfig /path/to/project
cargo run -- map --stylelint /path/to/project
cargo run -- map --prettier /path/to/project
cargo run -- map --cspell /path/to/project
cargo run -- map --npmrc /path/to/project
cargo run -- map --jscpd /path/to/project

# Without flags: shows general project structure
cargo run -- map /path/to/project
```

Output is always JSON. Example:
```json
{
  "tool": "clippy",
  "resolution": "walk-up from CARGO_MANIFEST_DIR — nearest clippy.toml wins, shadows completely",
  "project": "/path/to/project",
  "configs": [
    {
      "path": "apps/validator-rust/clippy.toml",
      "details": { "methods": { "total": 25, "required_total": 33, "required_present": 25, "required_missing": 8, "user_extra": 0 }, ... },
      "covers": ["apps/validator-rust"],
      "shadowed_by": []  // only if shadows exist
    }
  ],
  "uncovered": ["packages/low-expectations", "packages/seo-site-files"],
  "summary": { "total_dirs": 32, "covered_dirs": 25, "uncovered_dirs": 7 }
}
```
