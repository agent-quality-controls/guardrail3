# Generic coverage map engine

**Date:** 2026-03-19 09:53
**Scope:** coverage/engine.rs, coverage/clippy.rs, coverage/deny.rs, coverage/rustfmt.rs

## Summary
Extracted common walk-up resolution and rendering into a generic engine. Each tool implements `CoverageTool` trait. clippy/deny/rustfmt now use the shared engine. deny fixed to use proper per-crate walk-up (was incorrectly scope-level only).

## Decisions Made
- **CoverageTool trait:** name, resolution_description, config_files, targets, parse_details, walks_up. Engine handles all resolution + rendering.
- **Two-perspective output:** JSON shows both `configs` (each config file + what it covers) and `targets` (each crate + what covers it).
- **Shared rust_crate_targets:** clippy.rs exports the target builder, deny.rs and rustfmt.rs reuse it.

## Key Files for Context

- `apps/guardrail3/src/commands/coverage/clippy.rs` — clippy impl + shared rust_crate_targets
- `.plans/by_file/tools/` — tool docs with verified edge cases
