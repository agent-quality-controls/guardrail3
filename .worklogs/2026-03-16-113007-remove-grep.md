# Remove all grep fallbacks — AST-only source scanning

**Date:** 2026-03-16 11:30
**Scope:** All source scan modules (rs + ts), ast_helpers, ast_visitors

## Summary
Deleted all 15 grep fallback functions, grep supplement, and helpers. Source scanning is 100% AST-based. Net -1,344 lines. 353 tests passing, 0 failures.

## Context & Problem
Adversarial agents found that nearly all bugs were in grep helper code (string tracking, comment tracking, brace counting). AST paths had zero bugs. Grep fallbacks handled unparseable files, but unparseable files can't compile so the coverage is moot.

## Decisions Made
- No grep fallback: parse fails → skip file
- No macro body supplement: known AST limitation, accepted
- BOM stripping added to parse_file
- cfg(test) import skip added to find_std_fs_imports
