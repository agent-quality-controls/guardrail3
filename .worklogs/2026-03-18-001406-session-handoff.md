# Session Handoff — March 17-18, 2026

## What guardrail3 IS

A Rust CLI (`cargo install guardrail3`) that validates code project configuration and architecture. It does NOT re-implement linters. It ensures that linters, build tools, and architectural patterns are correctly configured so THEY catch violations at compile time or pre-commit.
