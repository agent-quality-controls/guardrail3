# Summary

Published the two G3TS npm packages and documented the working npm publish procedure.

The previous diagnosis was wrong: the npm token was not blocked. The failure came from using `npm whoami` as an auth gate and from relying on npmrc environment interpolation. The token publishes successfully when written literally into a temporary npmrc.

# Decisions Made

- Use a temporary npmrc for publish commands and delete it immediately after publish.
- Write the token value into the temporary npmrc from `NPM_TOKEN`.
- Do not treat `npm whoami` returning `E401` as proof that this token cannot publish.
- Keep the publish instructions in `NPM_PUBLISHING.md`.
