# Wire inline std::fs syn visitor into R58

**Date:** 2026-03-16 11:53
**Scope:** ast_helpers.rs, code_quality_checks.rs

## Summary
Added InlineStdFsVisitor to ast_helpers.rs and wired find_inline_std_fs_calls into check_direct_fs_usage. R58 now catches both `use std::fs` imports AND inline `std::fs::read_to_string(...)` calls — all via syn, zero grep. Un-ignored 3 R58 tests.

## Decisions Made
- Exclude type references (std::fs::Permissions, std::fs::Metadata) by checking if the 3rd path segment starts with lowercase (function) vs uppercase (type)
- Skip cfg(test) modules and functions via is_cfg_test_attr
- Deduplicate results when both use-import and inline-call detection find the same line
