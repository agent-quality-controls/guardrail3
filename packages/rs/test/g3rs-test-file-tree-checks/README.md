Runs the `test` family structural file-tree checks on one root-scoped test bundle.

Current rules:

- `RS-TEST-02` owned sidecar shape only
- `RS-TEST-03` runtime/assertions split and boundary enforcement
- `RS-TEST-10` required structural inputs fail closed
- `RS-TEST-18` `test_support` stays generic
